# validation-app-practice

## 概要

COACHTECH 教材 Tutorial 9-6「バリデーション ハンズオン演習」で作成した成果物です。
ユーザー登録フォームのバリデーションを作成しました

## 使用技術

- PHP 8.x
- Laravel 10.x
- FormRequest / バリデーションルール
- Git Hub

## 学んだこと

- コントローラーの肥大化を防ぐためにリクエストフォームクラスを作成してバリデーションルールを定義する仕方を学びました。

- messeges()でカスタムエラーメッセージの設定をできるようになりました。

-　コントローラーでvalidated()メソッドを使用して検証済みのデータを取り出して新規ユーザーを作成する方法を学びました。
